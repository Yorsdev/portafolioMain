03. Projects

Rick And Morty App with oficial API

In this project, I developed an interactive web application that consumes the official Rick and Morty API to dynamically display data about characters, locations, and episodes from the series.

Key Features

  • Smart search by character name or ID
  • Filters by status, species, and gender
  • Individual character details with a responsive design
  • Smooth navigation between views using React Router
  • Efficient state management and conditional rendering

Technologies Used

  • React 19
  • JavaScript (ES6+)
  • Fetch API
  • Custom CSS with a Mobile First aproach
  • API: Rick and Morty API(rickandmortyapi.com)

Pokédex Web App

This project is a dynamic and interactive web application that leverages the PokeAPI to explore the entire Pokémon universe across all generations, attributes and types.

Key Features

  • Browse all pokémons organized by generation
  • Detailed Pokémon cards showing types, stats, abilities, and evolution chains
  • Fully responsive design for all screen sizes
  • Smooth navigation between views using React Router
  • Smooth transitions and optimized data loading for a seamless experience.

Tech Stack

  • React + TypeScript
  • HTML5 & CSS3
  • TailwindCSS for fast and consistent styling
  • JavaScript for interactive behavior
  • RESTful API integration via Fetch
  • API: PokeAPI API(pokeapi.co)

Weather App with OpenWeather API

This project is a responsive and intuitive weather application that consumes real-time data from the OpenWeather API, allowing users to search for the current weather and forecast by city name.

Key Features

  • Search functionality by city name
  • Displays current weather, temperature, humidity, and wind speed
  • 5-day forecast with daily temperature and weather conditions
  • Dynamic icons and background changes based on weather conditions
  • Fully responsive layout optimized for all devices

Tech Stack

  • HTML5 & CSS3
  • JavaScript (ES6+) for data handling and interactivity
  • React + Vite
  • Fetch API to communicate with OpenWeather
  • API: OpenWeather(openweathermap.org)
