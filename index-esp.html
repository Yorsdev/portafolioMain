01. Conóceme

Soy un desarrollador web Fullstack con una pasión por la construcción de sitios web y aplicaciones modernas y sensibles. Tengo experiencia tanto en desarrollo front-end como back-end, y siempre estoy dispuesto a aprender nuevas tecnologías y mejorar mis habilidades. Soy una persona muy motivada y orientada al detalle, con una fuerte ética de trabajo y un compromiso con la excelencia. 02. ¿Qué se hacer?

Como desarrollador Fullstack, me especializo en tecnologías front-end y back-end, incluido HTML5 para estructurar contenido web, CSS3 para diseño responsivo y visualmente atractivo y desarrollo del lado del servidor.

HTML

Escribo HTML limpio y semántico para crear contenido bien estructurado y páginas web accesibles. Me concentro en las mejores prácticas para garantizar compatibilidad entre diferentes navegadores y dispositivos.

CSS3

Desde diseños hasta animaciones, uso CSS (incluidos Flexbox y Grid) para diseñar interfaces responsivas y visualmente atractivas. Disfruto haciendo que los sitios web se vean geniales en cualquier tamaño de pantalla.

Javascript

JavaScript es mi lenguaje preferido para agregar interactividad a los sitios web. Trabajo con características modernas de ES6+ para escribir código eficiente y fácil de mantener.

React

Utilizo React para crear aplicaciones front-end dinámicas y escalables. Disfruto trabajando con componentes, gestión de estados y ganchos para crear experiencias de usuario fluidas.

MySQL

Trabajo con MySQL para diseñar y gestionar bases de datos de manera eficiente. Escribir consultas, optimizar el rendimiento y garantizar la integridad de los datos son partes clave de mi desarrollo de backend.

03. Conoce mi trabajo

Landing page con animaciones (Vanilla)

Una landing page moderna y responsiva con animaciones sutiles, efectos de desplazamiento (scroll effects) y una UI atractiva. Tecnologías: HTML, CSS (Flexbox/Grid, Animaciones), JavaScript (GSAP o Intersection Observer).

Dashboard Interactivo con React

Un panel de control con gráficos dinámicos, filtros y un sistema de autenticación (simulado o real). Tecnologías: React, React Router, Hooks, Chart.js o Recharts para gráficos.

API REST con Node.js y MySQL

Un backend que maneja usuarios, autenticación con JWT y CRUD para un sistema de tareas. Tecnologías: Node.js, Express, MySQL, Sequelize, JWT para autenticación.

Ecommerce con React y MySQL

Un sistema básico de e-commerce con productos, carrito de compras y pedidos. Tecnologías: React (frontend), Node.js con Express (backend), MySQL (base de datos).
