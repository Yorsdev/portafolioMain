02. ¿Qué se hacer?

Como desarrollador Fullstack, me especializo en tecnologías front-end y back-end, incluido HTML5 para estructurar contenido web, CSS3 para diseño responsivo y visualmente atractivo y desarrollo del lado del servidor.

HTML

Escribo HTML limpio y semántico para crear contenido bien estructurado y páginas web accesibles. Me concentro en las mejores prácticas para garantizar compatibilidad entre diferentes navegadores y dispositivos.

CSS3

Desde diseños hasta animaciones, uso CSS (incluidos Flexbox y Grid) para diseñar interfaces responsivas y visualmente atractivas. Disfruto haciendo que los sitios web se vean geniales en cualquier tamaño de pantalla.

Javascript

JavaScript es mi lenguaje preferido para agregar interactividad a los sitios web. Trabajo con características modernas de ES6+ para escribir código eficiente y fácil de mantener.

React

Utilizo React para crear aplicaciones front-end dinámicas y escalables. Disfruto trabajando con componentes, gestión de estados y ganchos para crear experiencias de usuario fluidas.

MySQL

Trabajo con MySQL para diseñar y gestionar bases de datos de manera eficiente. Escribir consultas, optimizar el rendimiento y garantizar la integridad de los datos son partes clave de mi desarrollo de backend.

03. Conoce mi trabajo

Rick y Morty con API oficial (Frotend)

En este proyecto, desarrollé una aplicación web interactiva que consume la API oficial de Rick and Morty para mostrar dinámicamente datos sobre personajes, ubicaciones y episodios de la serie.

Características Principales

  • Búsqueda por nombre o ID
  • Filtrado por estado, especia y género
  • Detalles individuales del personaje con diseño responsivo
  • Navegación suave y fluida gracias a React Router
  • Manejo eficiente de estados y renderizado

Technologias usadas

  • React 19
  • JavaScript (ES6+)
  • Fetch API
  • CSS con enfoque a Mobile First
  • API: Rick and Morty API(rickandmortyapi.com)

BancaDev - API REST para servicios bancarios

Este proyecto representa una API backend robusta para sistemas bancarios digitales. Permite gestionar usuarios, controlar accesos por rol, autenticar de forma segura y realizar transferencias internas, ideal tanto para entornos académicos como aplicaciones reales.

Características Principales

  • Sistema de autenticación segura con JWT y contraseñas encriptadas
  • Gestión de roles y permisos (Administrador y Cliente) con validaciones por middleware
  • Transferencias entre cuentas de usuarios con historial de transacciones
  • Arquitectura modular y escalable con dominios organizados
  • Envío automatizado de correos de bienvenida con Nodemailer y plantillas Pug

Technologias usadas

  • Node.js + Express
  • TypeScript
  • PostgreSQL + TypeORM
  • JWT + bcryptjs
  • Nodemailer + Pug
  • Postman como herramienta de documentación y pruebas

Pokédex

Este proyecto es una aplicación web dinámica e interactiva que aprovecha PokeAPI para explorar todo el universo Pokémon a través de todas las generaciones, atributos y tipos.

Características principales

  • Búsqueda de pokémons organizados por generación
  • Estadísticas completas de cada pokémon
  • Diseño responsivo
  • Navegación sueave entre vistas usando React Router
  • Transiciones suaves y optimizadas para mejor experiencia

Teccnologías usadas

  • React + TypeScript
  • HTML5 & CSS3
  • TailwindCSS para un estilado rápido y consistente
  • JavaScript para un comportamiento interactivo
  • Integración de APIRest con fetch
  • API: PokeAPI API(pokeapi.co)

Aplicación del clima con API OpenWeather

Este proyecto es una aplicación meteorológica intuitiva y responsiva que consume datos en tiempo real de la API OpenWeather, lo que permite a los usuarios buscar el clima actual y el pronóstico por nombre de ciudad.

Características principales

  • Búsqueda funcional por nombre de ciudad
  • Visualización de clima actual, temperatura, humedad y velocidad del viento
  • Pronóstico de 5 días con temperatura diaria y condiciones climáticas
  • Los iconos dinámicos y el fondo cambian según las condiciones climáticas.
  • Diseño totalmente adaptable y optimizado para todos los dispositivos.

Tecnologías usadas

  • HTML5 & CSS3
  • JavaScript (ES6+) para el manejo de datos y la interactividad
  • React + Vite
  • Fetch API para comunicarse con OpenWeather
  • API: OpenWeather(openweathermap.org)
